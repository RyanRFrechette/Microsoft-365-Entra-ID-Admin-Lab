# Case Study: Microsoft 365 / Entra ID Administration

**Status:** Draft — to be completed in Phase 4.

## Scenario

A small organization migrates to Microsoft 365 Business Premium. As the sole IT admin, you are responsible for provisioning users, enforcing security baselines, managing group access, and responding to help desk tickets — all within Microsoft Entra ID and the M365 admin center.

## Objectives

- Provision new users with correct licenses and group memberships
- Enforce MFA for all accounts
- Respond to and document six realistic support tickets
- Automate repetitive tasks with PowerShell (Microsoft.Graph)
- Produce a Secure Score baseline report

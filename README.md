# Microsoft 365 / Entra ID Admin Lab

> **Audience:** HR professionals and technical recruiters evaluating IT support / cloud admin candidates.

## What This Lab Proves

This hands-on lab demonstrates real Microsoft 365 and Entra ID administration skills performed inside a live Business Premium tenant — not a simulation. Every ticket, script, and screenshot maps to a task that comes up daily in enterprise IT support and cloud administration roles.

| Skill Area | Evidence |
|---|---|
| User lifecycle management | Onboarding & offboarding tickets + screenshots |
| Identity & access control | Group creation, MFA enforcement, Conditional Access |
| License & app provisioning | Assigning M365 licenses and app permissions |
| Security operations | MFA review, sign-in log analysis, security reports |
| PowerShell automation | Scripts for bulk user and group tasks |
| Ticketing & documentation | Six structured support tickets with resolution notes |

## Tools Used

- **Microsoft Entra admin center** — user, group, role, and MFA management
- **Microsoft 365 admin center** — license assignment, app access
- **Azure portal** — Conditional Access, sign-in logs
- **PowerShell (Microsoft.Graph module)** — automation and reporting
- **Microsoft Secure Score** — security posture review

# Resume Bullets - Microsoft 365 / Entra ID Admin Lab

## Resume Bullets

- Built a Microsoft 365 / Entra ID admin support lab using a Business Premium tenant to demonstrate user lifecycle management, license assignment, group access, password reset, sign-in support, and offboarding workflows.
- Created and managed Microsoft 365 test users with Microsoft Graph PowerShell, including profile configuration, usage location updates, license assignment, password reset, sign-in blocking/restoration, and offboarding verification.
- Managed Entra ID security group access by creating a security group, adding a user, verifying membership in PowerShell and the Entra admin center, and removing access during offboarding.
- Exported Microsoft 365 user and license reports with Microsoft Graph PowerShell, including baseline user inventory, license inventory, and final offboarding audit reports.
- Documented six realistic Microsoft 365 support tickets covering onboarding, password reset/sign-in support, group access, license/app access, authentication methods review, and user offboarding.
- Created a hiring-manager-friendly GitHub portfolio repo with inline screenshots, support tickets, CSV reports, a case study, and PowerShell-based evidence of Microsoft 365 administration.
